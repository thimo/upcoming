# Upcoming

A macOS menu bar calendar. Month grid with per-calendar dots, an agenda
list that starts at today, and a join button for video calls — fed entirely
by the calendars you already sync in Calendar.app (iCloud, Microsoft 365 /
Exchange, Google, …). No accounts, no OAuth, read-only.

## Build

```sh
./build.sh
```

Builds, tests, signs, and installs to `~/Applications/Upcoming.app`.
Requires macOS 14+ and the Xcode Command Line Tools.
